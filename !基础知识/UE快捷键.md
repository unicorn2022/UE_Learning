# 一、动画蓝图

**双击连线**：连线添加一个结

**C+点击**：注释

**S+点击**：序列

**B+点击**：分支

**O+点击**：执行一次

**P+点击**：事件开始执行

**ctrl+拖出变量**：获取变量

**alt+拖出变量**：设置变量

**ctrl+w**：复制一个节点

# 二、视图

**ctrl+空格**：内容浏览器侧滑菜单

**ctrl+L**：设置太阳光的角度

**alt+鼠标左键**：绕某一物体旋转

**选中某一物体+F**：聚焦到该物体上

**在某一正交视图中，按住鼠标中键**：测量距离

# 三、材质蓝图

## 3.1	常用节点

**1/2/3+点击**：添加一/二/三维向量

**M+点击**：添加乘法节点

**U+点击**：添加**TexCoord**节点，用于UV平铺，与贴图的UVs相连

**选中多个节点+C**：添加注释

## 3.2	线性插值

**L+点击**：添加**Lerp**节点，线性插值节点

## 3.3	顶点相关属性

**右键|VertexColor**：添加**VertexColor**节点，顶点颜色节点，用于顶点绘制

**右键|VertexNormalWS**：添加**VertexNormalWS**节点，获取顶点法线
